## How to run 
```shell
robot --variable  SSH_HOST:[ADD_HOST_HERE] SSH_PORT:[ADD_PORT_HERE] SSH_USERNAME:[ADD_USERNAME_HERE] SSH_PASSWORD:[ADD_PASSWORD_HERE] production.robot 

```
Example
```shell
robot --variable  SSH_HOST:6.tcp.ngrok.io SSH_PORT:18268 SSH_USERNAME:massbit SSH_PASSWORD:[ADD_PASSWORD_HERE] production.robot 
```
